## Common Library

Used by client and server

### Link to project (development mode)

To link local _fluvio-lib_:

```
> npm link ../fluvio-lib/
```

Unlink

```
> npm link ../fluvio-lib/
```

### Install in project

To install local _fluvio-lib_

```
> npm install --save ../fluvio-lib/
```

Uninstall 

```
> npm uninstall ../fluvio-lib/
```


### Build Project

```
> npm run prepare
```
